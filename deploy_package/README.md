# تعليمات رفع التحديث على السيرفر

## الملفات المطلوب رفعها:

### 1. تشغيل الـ Migration (مرة واحدة فقط)
قبل رفع الملفات، قم بتشغيل هذا الـ SQL على قاعدة البيانات MySQL:
```
migration_new.sql
```
هذا الملف ينشئ جدول `barcode_serials` الجديد المطلوب لطباعة الستيكرات المتسلسلة.

### 2. رفع الملفات على السيرفر
ارفع المجلدات والملفات التالية إلى مجلد httpdocs على السيرفر:
- `dist/` (الكود المبني - استبدل المجلد القديم بالكامل)
- `web.config` (استبدل الملف القديم)

### ملاحظات مهمة:
- لا تحذف ملف `.env` الموجود على السيرفر
- لا تحذف مجلد `uploads/` الموجود على السيرفر
- بعد الرفع، أعد تشغيل الـ Application Pool في IIS
